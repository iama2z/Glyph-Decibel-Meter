# Glyph Decibel Meter

An Android app that visualizes audio decibel levels using the Nothing Phone's Glyph Matrix LEDs, creating a filling effect from bottom to top.

## Status
✅ Scaffolding and core logic implemented!

## Current Features
- Real-time decibel monitoring using `AudioRecord`.
- Dynamic UI with dB readout.
- Placeholder for Nothing Glyph SDK integration.

## Files Created
- `app/src/main/java/com/nothing/glyphdecibelmeter/MainActivity.kt`: Core logic for audio processing.
- `app/src/main/res/layout/activity_main.xml`: Main application UI.
- `app/src/main/AndroidManifest.xml`: App configuration and permissions.
- `build.gradle` & `settings.gradle`: Build and dependency configuration.
